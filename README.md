# MageAustralia_AiReports

AI-driven natural-language reporting for the Maho admin.

**Status:** v1 in development. See `specs/maho-module-ai-reports/2026-05-06-initial.md` in the monorepo for the design spec.

**Requires:** Maho (PHP 8.3+) with the `Maho_Ai` core module configured.

## Run unit tests

```bash
composer install
./vendor/bin/phpunit
```
